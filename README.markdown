# WarOS

## What is WarOS ##

**WarOS** is a (or will be a) game based on Arkanoid.

## How to play ##

Firstly, You need to install gosu gem:

      sudo gem install gosu

With the Gosu gem installed just run the warOS.rb file of the root directory:

      ./warOS.rb
